# 改善執行記錄顯示格式

## 問題分析

目前執行記錄表格中的「結果/錯誤」欄位顯示時文字擠在一起，難以閱讀。主要問題：

1. 模態框寬度只有 800px，可能不夠寬
2. 表格欄位寬度沒有合理分配
3. 結果訊息格式需要優化（間距、背景、列表樣式）

## 解決方案

### 1. 調整模態框和表格寬度

- 將執行記錄模態框的 `max-width` 從 `800px` 增加到 `1200px`
- 為執行記錄表格設置 `table-layout: fixed` 並分配欄位寬度

### 2. 優化結果訊息顯示格式

- 為結果訊息添加背景色和內邊距，提升可讀性
- 改善文字間距和行高
- 使用更好的列表樣式（例如每行項目前有清晰的標記）
- 確保換行符號正確顯示

### 3. 調整欄位寬度分配

- 執行時間：15%
- 狀態：10%
- 開始時間：15%
- 完成時間：15%
- 結果/錯誤：45%（給予更多空間顯示詳細訊息）

## 修改文件

- `frontend-church/src/views/admin/ScheduledJobs.vue`
- 調整模態框寬度（第 82 行）
- 為執行記錄表格添加欄位寬度樣式
- 優化結果訊息的顯示格式（第 108-116 行）
- 添加新的 CSS 樣式類別
