# 首頁優化建議

## 目前首頁結構

1. **Hero 區塊** - 歡迎訊息 + 兩個按鈕
2. **聚會時間** - 晚崇、早崇兩個卡片
3. **最新消息** - 3個公告卡片
4. **最新活動** - 3個活動卡片

## 優化建議

### ✅ 建議保留（核心功能）

#### 1. Hero 區塊
- **保留**：這是首頁的第一印象，很重要
- **優化建議**：
  - 可以考慮加入教會照片或背景圖片
  - 按鈕可以調整為更符合教會特色的文字

#### 2. 聚會時間
- **保留**：這是訪客最需要的基本資訊
- **優化建議**：
  - 可以考慮加入「本週主日信息」的快速預覽
  - 可以加入「線上聚會連結」（如果有）

#### 3. 重要通知（原「最新消息」）
- **保留**：剛新增，很重要
- **已優化**：只顯示置頂公告，避免與「最新活動」重複
- **已實作**：改為「重要通知」區塊，只顯示置頂的

#### 4. 緊急代禱事項
- **保留**：已新增，很重要
- **已實作**：只顯示標記為「緊急」的代禱事項（最多3個）
- 使用醒目的樣式提醒會友

#### 5. 最新活動
- **保留**：這是會友參與的重要資訊
- **優化建議**：
  - 可以考慮顯示更多（4-6個），因為活動通常較多
  - 或者改為「即將到來的活動」，只顯示未來一週的

#### 3. 最新主日信息（可選）
**位置**：在「聚會時間」之後

**理由**：
- 主日信息是教會的核心內容
- 可以吸引會友回顧或預習

**實作方式**：
- 顯示最新一則主日信息
- 包含：講題、講員、經文、日期
- 提供「查看全部主日信息」連結

### 🔄 建議調整

#### 1. Hero 區塊按鈕優化
**目前**：
- 「查看最新活動」
- 「第一次來訪？聯絡我們」

**建議**：
- 「查看最新活動」→ 保留
- 「第一次來訪？聯絡我們」→ 改為「了解更多」或「加入我們」

或者改為三個按鈕：
- 「查看最新活動」
- 「了解教會」
- 「聯絡我們」

#### 2. 最新消息 vs 最新活動的平衡
**問題**：兩者可能顯示相似的內容（如聖誕節活動）

**建議方案 A**（推薦）：
- **最新消息**：只顯示置頂公告（1-2個），改為「重要通知」
- **最新活動**：顯示更多活動（4-6個）

**建議方案 B**：
- **最新消息**：顯示所有公告（3個）
- **最新活動**：只顯示即將到來的活動（未來一週內，最多3個）

#### 3. 聚會時間區塊優化
**建議**：
- 可以加入「本週主日信息」的快速預覽
- 可以加入「線上聚會」連結（如果有）
- 可以加入「地圖導航」按鈕

### ❌ 不建議移除

所有目前的功能都有其價值，不建議移除任何區塊。

## 推薦的首頁結構（優化版）

### 方案 A：完整版（適合內容豐富的教會）

1. **Hero 區塊** - 歡迎訊息 + 3個按鈕
2. **聚會時間** - 晚崇、早崇 + 本週主日信息預覽
3. **重要通知** - 只顯示置頂公告（1-2個）
4. **緊急代禱** - 緊急代禱事項（1-3個）
5. **最新活動** - 即將到來的活動（4-6個）

### 方案 B：精簡版（適合內容較少的教會）

1. **Hero 區塊** - 歡迎訊息 + 2個按鈕
2. **聚會時間** - 晚崇、早崇
3. **重要通知** - 置頂公告（最多3個）
4. **緊急代禱** - 緊急代禱事項（最多3個）
5. **最新活動** - 即將到來的活動（3個）
